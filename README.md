# mindful-muse-finder-updated

Welcome — this repository contains the source for the Mindful Muse Finder (a Vite + React + TypeScript project using shadcn-ui and Tailwind).

## Project info

- Local dev: `npm install` then `npm run dev`
- Build: `npm run build`

## How to edit locally

1. Ensure Node.js and npm are installed (nvm recommended).
2. Clone and run the project:

```sh
git clone https://github.com/som21-star/mindful-muse-finder-updated.git
cd mindful-muse-finder-updated
npm install
npm run dev
```
